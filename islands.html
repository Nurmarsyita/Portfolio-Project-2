    • Islands in Singapore

    Overview

    Our task was to create a scrollable list containing different types of islands off-shore in
    Singapore. Upon clicking the button to view more details about a certain island, the user will
    be directed to another page containing the island's details. A key feature of the application I
    created is that the fonts can be resized by the user.

    Key Feature

    Fonts can be resized by the user
